Title:
Multi-Game Video Poker
Requires g++ and terminal access
Installation:
 g++ main.cpp
Usage: 
./a.out to start, then use the number pad to select options on the menus.
Includes:
 Voucher system for inserting credits into the game
Ability to cash out your credits to voucher.txt
Video Poker Game with full functionality
A number guessing game for earning credits if you go broke
Structure:
main.cpp - main menu and game select
poker.h - handles the poker menu and gamestate
LuckyGuess.h - handles the Lucky Guess game
General.h - general functions
voucher.txt - a text file with a single floating point number with up to 2 decimal places
Have fun!! :)
